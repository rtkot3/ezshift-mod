# r3 Extension for EZShift 🗓️

![Version](https://img.shields.io/badge/version-3.2-purple)
![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-EZShift-blue)

**r3** — это продвинутый UserScript для автоматизации и расширения возможностей интерфейса **EZShift**. Скрипт предназначен для сотрудников, которым требуется быстрый поиск коллег, анализ команд и выявление сотрудников вне их основных зон ответственности.
